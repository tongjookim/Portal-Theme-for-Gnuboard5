# Portal Theme for Gnuboard5

네이버/다음 스타일의 **포털형 레이아웃**을 제공하는 그누보드5(Gnuboard5) 테마입니다. 웹표준 및 접근성을 준수하며, 코드 수정 없이 관리자 화면에서 로고·배너·위젯·색상·다크모드까지 직접 편집할 수 있습니다.

- Theme URI: http://www.maru.or.kr/
- Maker: 수완뉴스 (https://www.maru.or.kr/)
- License: GNU Lesser General Public License v2.1 (`LICENSE` 참고)

## 주요 특징

- **포털형 2단 레이아웃**: 상단 유틸리티바 + 헤더(로고/검색/헤더배너) + GNB + 좌측 콘텐츠/우측 위젯 구조의 홈 화면
- **관리자 편집 모달**: 최고관리자로 로그인 시 홈 화면에서 로고 / 메인 배너(슬라이더) / 헤더배너 / 푸터 / 위젯 / 광고 / 색상을 탭으로 전환하며 실시간 편집·저장 (별도 설정 페이지 이동 불필요)
- **테마 포인트 컬러 프리셋**: 기본(그린) / 브릭레드 / 포레스트그린 / 스카이블루 / 네이비 5종, CSS 커스텀 프로퍼티 기반이라 프리셋 전환 시 전체 UI(코어 스킨 포함)에 즉시 반영
- **다크모드**: 헤더 우측 토글로 라이트/다크 전환, `localStorage` 저장 + FOUC(깜빡임) 방지 스크립트 적용, 핵심 레이아웃(헤더/GNB/위젯/게시판 목록·보기/내용관리 등) 대응
- **회원별 노출 제외**: 광고·헤더배너 등을 특정 회원(`mb_id`)에게는 노출하지 않도록 설정 가능
- **다양한 게시판 스킨**: 기본형(`list`) 외에 네이버 카페 스타일(`cafe_style`), 갤러리형(`gallery`), 매거진형(`webzine`) 스킨 포함 (PC/모바일 각각 대응)
- **이미지 업로드 검증**: 로고(SVG 새니타이징), 배너 등 업로드 시 확장자 화이트리스트·용량 제한·실제 이미지 검증을 거쳐 안전하게 저장
- **통합검색 연동**: 통합검색 플러그인이 설치되어 있으면 연동하고, 미설치 시 기본 검색 스킨으로 자동 폴백
- **ToTop 버튼**: 스크롤 시 노출되는 맨 위로 이동 버튼 (포인트 컬러 프리셋과 연동)

## 설치 방법

1. 그누보드5 사이트의 `theme` 디렉토리로 이동하여 이 저장소를 `portal`이라는 이름으로 클론(또는 압축 해제)합니다.
   ```bash
   cd /경로/그누보드설치폴더/theme
   git clone https://github.com/tongjookim/Portal-Theme-for-Gnuboard5.git portal
   ```
2. `data/` 디렉토리에 웹서버 쓰기 권한을 부여합니다. (테마 설정·업로드 이미지가 저장되는 경로이며, 저장소에는 커밋되지 않습니다 — `.gitignore` 참고)
   ```bash
   chmod -R 707 theme/portal/data
   ```
3. 그누보드5 관리자 페이지 → **환경설정** → **기본환경설정**에서 테마를 `portal`로 지정하고 저장합니다.
4. 최고관리자 계정으로 사이트 홈에 접속하면 우측 하단(또는 상단)에 테마 편집 진입점이 노출됩니다. 편집 모달에서 로고 / 배너 / 헤더배너 / 푸터 / 위젯 / 광고 / 색상을 설정합니다.
5. (선택) 메뉴 관리에서 GNB에 노출할 메뉴를 등록합니다. 메뉴가 없으면 관리자에게 등록을 안내하는 문구가 대신 출력됩니다.

### 요구 사항

- Gnuboard5 (최신 버전 권장)
- PHP 7.4 이상 (그누보드5 권장 사양 기준)
- 테마 설정을 변경하려면 최고관리자(`super`) 권한 필요
